selecao_capgemin
 

#Descrição
Projeto trata-se de questões avaliativas para processo seletivo, os projetos foram criados separadamente uma vez que a funções não se complementam, por isso foram criados separamente os projetos.  


#Requisitos para executar os projetos

   * JDK 8:Necessario para executar o projeto Java
   * IDE Eclipse: Para desenvolvimento do projeto
   

#Tecnologias utilizadas

Os projetos foram desenvolvidos na linguagem Java, aplicação console.Foram utilizado conceitos de lógica de programação, Classe, Objeto, estutura de repetição.
